Refuge-Android
==============

Android app for RefugeRestrooms.org

Play Store:
https://play.google.com/store/apps/details?id=com.sopinka.refuge

Requirements To Build
==============

- Android Studio (uses Gradle to build)
- Android 4.0 SDK

Tech
==============

- Android
- Java
- Gson
- Google Maps V2
- Google AdSense
- XML
- JSON
